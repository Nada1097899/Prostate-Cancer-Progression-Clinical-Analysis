#  Prostate Cancer Progression Analysis

A clinical data science project focused on predicting tumor volume levels in prostate cancer patients using advanced machine learning ensemble techniques.

## Project Overview
Analyzing clinical data from MSKCC (2020), this project builds a classification system to distinguish between High and Low tumor volumes. This automation supports precision medicine by helping clinicians assess disease progression more efficiently.

##  Tech Stack & Methods
* **Analysis:** Clinical Data Exploration (EDA), Handling Missing Data, Label Encoding.
* **Machine Learning:** AdaBoost, Gradient Boosting, Bagging, Random Forest, Decision Trees.
* **Evaluation:** Confusion Matrix, Precision-Recall, F1-Score.
* **Libraries:** Pandas, Seaborn, Scikit-learn, NumPy.

##  Technical Deep Dive
- **Data Integration:** Processed complex clinical metadata including Study ID, Patient ID, and specific clinical attributes.
- **Ensemble Benchmarking:** Systematically evaluated different boosting and bagging strategies to find the most stable model for medical data.
- **Diagnostic Performance:** Achieved balanced precision and recall, ensuring the model is reliable for identifying both high and low-risk cases.

##  Key Findings
- **AdaBoost Performance:** Showed superior ability in handling clinical feature variance compared to simpler models.
- **Clinical Correlation:** Identified key attributes that drive the classification of tumor progression.
